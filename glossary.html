Glossary

A

A Record - A DNS record type that maps a domain name to an IPv4 address. The most common DNS record type.

AAAA Record - A DNS record type that maps a domain name to an IPv6 address (128-bit).

Access Control - Security mechanisms that regulate who or what can view or use resources in a computing environment.

Active Directory Domain Services (AD DS) - Microsoft's directory service that manages users, computers, and other objects in a Windows domain network. Provides authentication, authorization, and centralized administration.

Advanced Forensic Format (AFF) - An open-source, extensible forensic image format supporting compression, encryption, metadata, and advanced features. Designed to be patent-free and vendor-neutral.

Adversary - An individual, group, organization, or government that conducts or has the intent to conduct detrimental activities against an organization or system.

API (Application Programming Interface) - A set of protocols, tools, and definitions that allows different software applications to communicate with each other.

APT (Advanced Persistent Threat) - A prolonged and targeted cyber attack in which an intruder gains access to a network and remains undetected for an extended period, often to steal data.

ARPANET - Advanced Research Projects Agency Network, the precursor to the modern internet, developed by the U.S. Department of Defense in the late 1960s.

Asset - Anything of value to an organization, including data, hardware, software, facilities, personnel, and reputation.

Attack - Any kind of malicious activity that attempts to collect, disrupt, deny, degrade, or destroy information system resources or the information itself.

Attack Surface - The total sum of vulnerabilities, pathways, and methods that an attacker could use to gain unauthorized access to a system or network.

Audit Trail - A chronological record of system activities that provides documentary evidence of sequences of activities affecting a specific operation or event.

Authentication - The process of verifying the identity of a user, device, or system. In cryptography, verifying that a user is who they claim to be.

Authoritative Name Server - The DNS server that has the actual DNS records for a specific domain. It provides the definitive answer for queries about that domain.

Automation - The use of technology to perform tasks with minimal human intervention, often through scripts or dedicated tools.

Availability - Ensuring that authorized users have reliable and timely access to information and resources when needed. One of the three pillars of the CIA Triad.

B

Backup - A copy of data stored separately from the original to protect against data loss from hardware failure, corruption, or deletion.

Biometrics - Authentication method using unique physical characteristics such as fingerprints, facial recognition, iris scans, or voice patterns.

Black Box - Something that you can't see how it works. In security testing, testing where the tester has no knowledge of the internal workings of the system.

Black Box Testing - Security testing where the tester has no knowledge of the internal workings of the system, simulating an external attack.

Blue Team - A group responsible for defending against attacks and improving security posture.

C

California Consumer Privacy Act (CCPA) - Similar provisions as GDPR for residents of California.

Chain of Custody (CoC) - A documented chronological record that tracks the handling, transfer, location, and possession of physical or electronic evidence from collection through presentation in court. Essential for maintaining evidence integrity and admissibility.

CIA Triad - The three foundational principles of information security: Confidentiality, Integrity, and Availability. Sometimes expanded to include Authentication, Authorization, Accounting, and Non-repudiation.

CLI (Command-Line Interface) - A text-based interface where users interact with the system by typing commands.

Clone - An exact duplicate of a storage device that can be used to replace the original device. Unlike an image, a clone is a physical device-to-device copy that can be booted and used operationally.

Cloud Computing - The delivery of computing services (servers, storage, databases, networking, software) over the internet ("the cloud").

CNAME Record (Canonical Name) - A DNS record that creates an alias for another domain name, allowing one domain to point to another.

Compliance - Adherence to laws, regulations, standards, and policies that apply to an organization's industry and operations.

Computer Fraud and Abuse Act (CFAA) - Anyone who engages in unauthorized computer access faces consequences ranging from fines to imprisonment.

Confidentiality - Using cryptography to keep data private. Ensuring that information is accessible only to those authorized to access it. One of the three pillars of the CIA Triad.

Containerization - A lightweight alternative to virtualization that packages applications and their dependencies together, allowing them to run consistently across different computing environments (e.g., Docker).

Contamination - Any alteration, addition, or deletion of evidence data that compromises its integrity and admissibility in legal proceedings.

Creeper - The first documented computer virus (1971), a self-replicating program that displayed the message "I'm the creeper, catch me if you can!"

Cryptographic Salt - A random string of characters added to a password before hashing to protect against rainbow table attacks and ensure unique hashes even for identical passwords.

CVE (Common Vulnerabilities and Exposures) - A publicly disclosed security vulnerability identified by a unique ID (e.g., CVE-2021-44228 for Log4Shell).

D

Data Breach - An incident where sensitive, protected, or confidential data is accessed, disclosed, or stolen by an unauthorized individual.

DDoS (Distributed Denial of Service) - A DoS attack that uses multiple compromised systems (often a botnet) to flood the target with traffic.

Defense in Depth - A security strategy that employs multiple layers of defense mechanisms to protect assets, so that if one layer fails, others continue to provide protection.

DHCP (Dynamic Host Configuration Protocol) - A network protocol that automatically assigns IP addresses and other network configuration parameters to devices on a network.

Digital Certificate - An electronic document that uses a digital signature to bind a public key with an identity (person or organization). Issued by Certificate Authorities (CAs).

Digital Signature - A cryptographic mechanism that provides authentication, integrity, and non-repudiation by using public key cryptography to "sign" data.

DMZ (Demilitarized Zone) - A physical or logical subnetwork that contains and exposes external-facing services to an untrusted network, usually the internet, while keeping the internal network secure.

DNS (Domain Name System) - A hierarchical distributed naming system that translates human-readable domain names (like www.google.com) into IP addresses that computers use to identify each other on networks.

DNS Caching - The temporary storage of DNS query results to speed up subsequent lookups and reduce network traffic.

Domain Controller - A Windows server running AD DS that authenticates users and computers in a Windows domain, enforces security policies, and stores the Active Directory database.

DoS (Denial of Service) - An attack that makes a system, network, or service unavailable to legitimate users by overwhelming it with traffic or exploiting vulnerabilities.

E

EnCase Expert Witness Format (EWF/E01) - A proprietary forensic image format that supports compression, encryption, metadata storage, hashing, and splitting files into manageable segments.

Encryption - The process of encoding information so that only authorized parties can access it, typically using cryptographic algorithms and keys.

Evidence Container - A file format designed to store forensic images along with metadata. Examples include E01 (EnCase) and AFF (Advanced Forensic Format).

Exploit - A piece of software, data, or sequence of commands that takes advantage of a vulnerability to cause unintended behavior in a system.

F

File Services - Windows Server role that provides technologies for storage management, file replication, distributed namespace management, and shared folder access.

Firewall - A network security device or software that monitors and controls incoming and outgoing network traffic based on predetermined security rules.

Forensic Image - A bit-by-bit copy of a storage device that captures all data including active files, deleted files, slack space, and unallocated space. Unlike a simple copy, it preserves metadata and can be validated using cryptographic hashes.

Forensics - The application of scientific methods to collect, preserve, analyze, and present evidence from digital devices in a manner acceptable in legal proceedings.

FTK (Forensic Toolkit) Format - A proprietary format from AccessData with similar features to EWF, supporting metadata, compression, encryption, and file splitting.

FTP (File Transfer Protocol) - A standard network protocol used to transfer files between a client and server over a network.

G

Gateway - A network node that serves as an access point to another network, often performing protocol translation.

General Data Protection Regulation (GDPR) - A law that protects the personal private data for citizens of the European Union.

Geofencing - Only allowing access from specific locations.

Gray Box Testing - Security testing where the tester has partial knowledge of the system, combining elements of white box and black box testing.

GUI (Graphical User Interface) - A user interface that allows interaction with electronic devices through graphical icons and visual indicators, as opposed to text-based command-line interfaces.

H

Hash Value - A unique fixed-length string generated by a cryptographic algorithm from data of any size. Used to verify data integrity. Also called a "fingerprint" or "digest."

Healthcare Insurance Portability and Accountability Act (HIPAA) - A law that protects the Protected Health Information (PHI).

HTTP (Hypertext Transfer Protocol) - The foundation of data communication on the World Wide Web, defining how messages are formatted and transmitted.

HTTPS (HTTP Secure) - An extension of HTTP that uses encryption (TLS/SSL) to secure communication between web browsers and servers.

Hypervisor - Software that creates and runs virtual machines by abstracting the underlying physical hardware.

I

IaaS (Infrastructure as a Service) - Cloud computing model providing virtualized computing resources over the internet (e.g., AWS EC2, Azure VMs).

ICANN - Internet Corporation for Assigned Names and Numbers. Manages the DNS root zone.

IDS (Intrusion Detection System) - A system that monitors network traffic for suspicious activity and alerts administrators to potential security breaches.

IETF - Internet Engineering Task Force, an open standards organization that develops and promotes voluntary internet standards, particularly those comprising the Internet Protocol Suite (TCP/IP).

Impact - The magnitude of harm that could result from a threat exploiting a vulnerability. Can be measured in financial, operational, or reputational terms.

Impersonation - Pretending to be someone else to gain unauthorized access to systems, information, or physical locations. A common physical security threat.

Incident - An event that violates or threatens to violate security policies, acceptable use policies, or standard security practices.

Incident Response - The systematic approach to managing and addressing security incidents, including preparation, detection, containment, eradication, recovery, and lessons learned.

Indicator of Compromise (IoC) - Forensic evidence that suggests a system has been breached or infected, such as unusual network traffic, file modifications, or suspicious registry changes.

Integrity - Using cryptography to ensure that data has not been altered. Maintaining the accuracy, consistency, and trustworthiness of data throughout its lifecycle. One of the three pillars of the CIA Triad.

Internet - Global network of computers.

IoT - Internet of Things, often in reference to "smart home" electronics.

IP Address (Internet Protocol Address) - A numerical label assigned to each device connected to a network that uses IP for communication. Can be IPv4 (32-bit) or IPv6 (128-bit).

IPS (Intrusion Prevention System) - An active security system that monitors network traffic and can automatically take action to block or prevent detected threats.

IPv4 - Version 4 of the Internet Protocol using 32-bit addresses (e.g., 192.168.1.1), providing approximately 4.3 billion unique addresses.

IPv6 - Version 6 of the Internet Protocol using 128-bit addresses (e.g., 2001:0db8:85a3::8a2e:0370:7334), providing vastly more unique addresses than IPv4.

ISP (Internet Service Provider) - A company that provides internet access to customers.

L

LAN (Local Area Network) - A network that connects computers and devices in a limited geographical area like a home, office, or building.

Least Privilege Principle - Giving people the minimum access required to do their jobs. A security concept requiring that users, processes, and systems be granted only the minimum levels of access necessary to perform their functions.

Live Acquisition - The process of collecting digital evidence from a powered-on, running system. Captures volatile data like RAM contents, running processes, and active network connections, but risks contamination.

Logical Acquisition - Copying specific files, folders, or partitions from a storage device rather than imaging the entire physical device.

M

MAC Address (Media Access Control Address) - A unique hardware identifier assigned to network interface cards, used for communication within a local network segment.

Malware - Short for "malicious software." Any software intentionally designed to cause damage, gain unauthorized access, or perform unwanted actions on a computer system.

McCumber Cube - Represents design aspects for securing data in every level of abstraction.

MD5 (Message Digest 5) - A cryptographic hash function that produces a 128-bit hash value. Widely used for integrity verification in forensics, though cryptographically broken for security purposes.

Multifactor Authentication - The use of a secondary device(s) or media(s) for authentication, on top of the typical username/password.

MX Record (Mail Exchange) - A DNS record that specifies the mail servers responsible for accepting email for a domain.

N

NAT (Network Address Translation) - A method of remapping IP addresses by modifying network address information in packet headers, commonly used to allow multiple devices on a private network to share a single public IP address.

Non-Repudiation - Ensure that user performed activity by proof that they cannot deny. Typically achieved through digital signatures and digital certificates.

Norman Doors - Confusing or misleading user experiences. Design elements that provide confusing or misleading cues about their operation, named after Don Norman. In security contexts, refers to poor user interface design that leads to user errors and potential security issues.

NSFNET - National Science Foundation Network, which replaced ARPANET in 1990 and later became the backbone of the modern internet before being decommissioned in 1995.

O

Open Systems Interconnection (OSI) Model - Network protocol framework that uses 7 layers.

Order of Volatility - The sequence in which digital evidence should be collected, based on how quickly the data will be lost. From most to least volatile: CPU cache, RAM, network connections, running processes, disk storage, backups.

OSINT - Open Source Intelligence.

P

PaaS (Platform as a Service) - Cloud computing model providing a platform allowing customers to develop, run, and manage applications without managing underlying infrastructure (e.g., Azure App Service, Heroku).

Packet - A unit of data transmitted over a network. Contains both the payload (actual data) and header information (addressing, routing, error checking).

Password Complexity - Requirements for passwords to include a mix of uppercase, lowercase, numbers, and special characters to increase security.

Password Manager - A tool to organize and secure one's passwords for online services. A tool that stores and organizes passwords in an encrypted database, allowing users to maintain unique, complex passwords for different services.

Patch - A software update designed to fix bugs, address security vulnerabilities, or improve functionality.

Payload - The component of malware that performs the malicious action, such as stealing data, encrypting files, or opening a backdoor.

Penetration Testing (Pen Testing) - Authorized simulated cyber attacks performed to evaluate the security of a system or network.

PHI (Protected Health Information) - Any information about health status, provision of healthcare, or payment for healthcare that can be linked to a specific individual. Protected under HIPAA.

Phishing - A social engineering attack using fraudulent emails, messages, or websites to trick victims into revealing sensitive information or installing malware.

Physical Acquisition - Creating a complete bit-by-bit copy of an entire storage device, including all sectors, whether allocated or not.

PII (Personally Identifiable Information) - Any data that could potentially identify a specific individual, including names, addresses, Social Security numbers, biometric data, etc.

Port - A numbered endpoint in a network connection used to identify specific processes or services. Examples: HTTP uses port 80, HTTPS uses port 443.

PowerShell - Microsoft's task automation and configuration management framework consisting of a command-line shell and scripting language built on .NET.

Principle of Least Privilege - A security concept requiring that users, processes, and systems be granted only the minimum levels of access necessary to perform their functions.

Protocol - A set of rules and standards that define how data is transmitted and received over a network. Examples: TCP, UDP, HTTP, HTTPS, DNS, DHCP.

PTR Record (Pointer Record) - A DNS record used for reverse DNS lookups, mapping an IP address back to a domain name.

Purple Team - A collaborative approach where red and blue teams work together to improve security through shared knowledge.

R

Ransomware - Malicious software that encrypts a victim's files or locks their system, demanding payment (ransom) to restore access. Example: WannaCry.

Raw Image (dd) - A bit-by-bit copy of a storage device with no compression, metadata, or error handling. File extension is typically .dd or .raw.

RBAC (Role-Based Access Control) - An access control method where permissions are assigned to roles rather than individuals, and users are assigned to appropriate roles.

RDP (Remote Desktop Protocol) - A proprietary protocol developed by Microsoft that provides a graphical interface for connecting to another computer over a network.

Red Team - A group that plays the role of attackers to test an organization's security defenses.

Resolving Name Server - A DNS server (also called a recursive resolver) that receives DNS queries from clients and performs the necessary lookups to return the corresponding IP address. Typically provided by ISPs or can be set manually (like Google's 8.8.8.8).

Restore - The process of recovering data from a backup to return systems or files to a previous state.

Risk - The potential for loss, damage, or destruction of assets as a result of a threat exploiting a vulnerability. Typically calculated as Risk = Threat × Vulnerability × Impact.

Root Domain Server - The top-level DNS servers in the DNS hierarchy that know the locations of all TLD (Top-Level Domain) name servers. There are 13 root server systems worldwide (labeled A through M).

Router - A networking device that forwards data packets between computer networks, determining the best path for data transmission.

S

SaaS (Software as a Service) - Cloud computing model where applications are hosted by a provider and made available to customers over the internet (e.g., Office 365, Salesforce).

Sandbox - An isolated testing environment that allows programs to run without affecting the system or other applications, used for testing potentially malicious code or new features.

Script - A program or sequence of instructions that is interpreted or executed by another program rather than compiled into machine code.

Security Awareness Training - Education programs designed to help users recognize and respond to security threats like phishing, social engineering, and malware.

Security by Design - The practice of building security into systems from the beginning of development rather than adding it later.

Server Core - A minimal installation option for Windows Server that runs without a graphical user interface (GUI), reducing attack surface and resource requirements.

SHA-1 (Secure Hash Algorithm 1) - A cryptographic hash function producing a 160-bit hash value. Previously standard but now deprecated for security use due to collision vulnerabilities. Still used in forensics for legacy compatibility.

SHA-2 - A family of cryptographic hash functions including SHA-224, SHA-256, SHA-384, and SHA-512. Currently recommended for security and integrity verification.

SIEM (Security Information and Event Management) - A system that aggregates and analyzes security data from across an organization's infrastructure to detect threats and manage incidents.

Single Sign-On (SSO) - An authentication scheme allowing users to log in once and gain access to multiple related systems without re-entering credentials.

Slack Space - The unused space between the end of a file and the end of the disk cluster it occupies. May contain remnants of previously deleted data.

SOC (Security Operations Center) - A centralized facility where security professionals monitor, detect, analyze, and respond to cybersecurity incidents.

Social Engineering - Psychological manipulation tactics used to trick people into divulging confidential information or performing actions that compromise security.

SRV Record (Service Locator) - A DNS record that specifies the location (hostname and port) of servers for specific services like SIP, LDAP, or XMPP.

SSH (Secure Shell) - A cryptographic network protocol for operating network services securely over an unsecured network, commonly used for remote command-line login.

Static Acquisition - The process of collecting digital evidence from a powered-off system. Typically involves removing the storage device and connecting it through a write-blocker.

Steganography - The practice of concealing information within another message or physical object to avoid detection. The practice of concealing messages or information within other non-secret data, such as hiding text within an image file.

Stuxnet - First documented virus with real world physical damage, targeted Iran nuclear facilities. A sophisticated computer worm discovered in 2010, notable for being the first documented malware designed to cause real-world physical damage.

Subnet (Subnetwork) - A logical subdivision of an IP network, allowing network administrators to divide a larger network into smaller, manageable segments.

Switch - A network device that connects devices within a network and uses MAC addresses to forward data to the correct destination.

T

Tailgating - A physical security breach where an unauthorized person follows an authorized person into a restricted area. Common physical security threat.

TCP (Transmission Control Protocol) - A connection-oriented protocol that ensures reliable, ordered delivery of data between applications over a network.

The Web - Collection of info accessed using internet.

Threat - Anything that can exploit a vulnerability, intentionally or accidentally, and obtain, damage, or destroy an asset.

Threat Intelligence - Information about current or emerging threats that helps organizations understand and mitigate risks.

TLD Name Server (Top-Level Domain) - DNS servers responsible for top-level domains like .com, .org, .edu, .gov, or country codes like .uk, .jp. They know the authoritative name servers for all domains under their TLD.

TLS/SSL (Transport Layer Security / Secure Sockets Layer) - Cryptographic protocols that provide secure communication over a network. SSL is the deprecated predecessor to TLS.

U

UDP (User Datagram Protocol) - A connectionless protocol that sends data without establishing a connection or guaranteeing delivery, used for time-sensitive applications like video streaming.

Unallocated Space - Areas of a storage device that are not currently assigned to any file. Often contains deleted files or fragments of deleted data.

Update - A newer version of software that may include patches, new features, performance improvements, or security enhancements.

URL (Uniform Resource Locator) - The address used to access resources on the internet, specifying the protocol, domain, path, and potentially query parameters.

V

Validation - The process of verifying that a forensic image is identical to the source evidence, typically using hash algorithms like MD5, SHA-1, or SHA-2.

Virtual Machine (VM) - A software emulation of a physical computer that runs an operating system and applications as if it were a physical machine.

Virtualization - The creation of virtual (rather than physical) versions of computing resources like servers, storage devices, or networks.

VPN (Virtual Private Network) - A technology that creates a secure, encrypted connection over a less secure network, such as the internet.

Vulnerability - Weakness in an information system, system security procedures, internal controls, or implementation that could be exploited or triggered by a threat source. A weakness in a system, application, or network that could be exploited by threats to gain unauthorized access or cause harm.

Vulnerability Assessment - The systematic process of identifying, quantifying, and prioritizing vulnerabilities in a system.

W

WAN (Wide Area Network) - A network that covers a broad area, connecting multiple LANs across cities, countries, or continents.

Wannacry - Infamous ransomware. A ransomware attack in May 2017 that infected over 200,000 computers across 150 countries, exploiting a Windows vulnerability and causing billions in damages.

White Box Testing - Security testing where the tester has full knowledge of the internal structure, design, and code of the system being tested.

World Wide Web - An information system where documents and resources are identified by URLs, interlinked by hypertext, and accessible via the internet. Invented by Tim Berners-Lee in 1991.

Write-Blocker - A hardware or software tool that allows read-only access to data storage devices, preventing any modification of evidence during forensic examination.

Z

Zero Trust - A security model based on the principle "never trust, always verify," requiring strict identity verification for everyone trying to access resources regardless of location.

Zero-Day Exploit - An attack that exploits a previously unknown vulnerability before the software vendor has released a patch.
